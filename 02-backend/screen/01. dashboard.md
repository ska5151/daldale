# 대시보드 API 문서

## 1. 개요
대시보드 화면에 필요한 요약 정보를 제공하는 API 명세서입니다.

## 2. API 목록

### 2.1 대시보드 요약 정보 조회
- **URL**: `/api/dashboard/summary`
- **Method**: `GET`
- **Description**: 지정된 연/월의 고정지출 요약 정보와 결제 수단별 통계 데이터를 반환합니다.

#### Request Parameters
| Parameter | Type | Required | Description | Default |
|---|---|---|---|---|
| year | integer | N | 조회 연도 (YYYY) | 현재 연도 |
| month | integer | N | 조회 월 (M or MM) | 현재 월 |

#### Response
- **Success (200 OK)**
```json
{
  "code": 200,
  "message": "조회 성공",
  "data": {
    "period": "2024-05",
```

#### Implementation Note
- **DB Query**: `dashboard.sql`의 View 또는 집계 쿼리를 활용합니다.
- **Paid Amount Logic**: `payment_day`가 현재 날짜(`DAY(CURRENT_DATE())`)보다 작거나 같으면 '지출 완료'로 간주하거나, 별도의 `is_paid` 플래그가 있다면 그것을 사용합니다. (현재 DB 설계상 `payment_day`로 판단 권장)
