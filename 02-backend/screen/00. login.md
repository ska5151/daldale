# 로그인/회원가입 API 문서

## 1. 개요
로그인 및 회원가입 처리를 위한 백엔드 API 명세서입니다.

## 2. API 목록

### 2.1 로그인
- **URL**: `/api/auth/login`
- **Method**: `POST`
- **Description**: 사용자 아이디와 패스워드로 로그인합니다. 성공 시 인증 토큰(JWT 등)을 반환합니다.

#### Request Body
| Field | Type | Required | Description |
|---|---|---|---|
| user_id | string | Y | 사용자 아이디 |
| password | string | Y | 사용자 패스워드 |

#### Response
- **Success (200 OK)**
```json
{
  "code": 200,
  "message": "로그인 성공",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR...", 
    "user": {
        "user_id": "user123",
        "nickname": "알뜰이",
        "profile_image": null
    }
  }
}
```

- **Error (401 Unauthorized)**
```json
{
  "code": 401,
  "message": "아이디 또는 패스워드가 올바르지 않습니다."
}
```

### 2.2 회원가입
- **URL**: `/api/auth/register`
- **Method**: `POST`
- **Description**: 신규 사용자를 등록합니다. 초기 닉네임은 아이디와 동일하거나 별도 설정 가능합니다.

#### Request Body
| Field | Type | Required | Description |
|---|---|---|---|
| user_id | string | Y | 사용자 아이디 |
| password | string | Y | 사용자 패스워드 |
| email | string | Y | 사용자 이메일 |

#### Response
- **Success (201 Created)**
```json
{
  "code": 201,
  "message": "회원가입 성공"
}
```

- **Error (400 Bad Request)**
```json
{
  "code": 400,
  "message": "입력 값이 유효하지 않거나 이미 존재하는 아이디입니다."
}
```
