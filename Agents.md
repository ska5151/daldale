# 에이전트

## 기술 스택
- **프론트엔드**: React
- **백엔드**: node.js
- **데이터베이스**: MySQL
- **스타일링**: Tailwind CSS

## 프로젝트 개요

사용자가 매월 반복되는 고정 지출(구독, 월세, 할부 등)을 효율적으로 관리하고 모니터링할 수 있는 웹 어플리케이션입니다.

### 주요 기능

1.  **로그인 및 회원가입**
    *   **로그인**: 아이디, 패스워드를 입력하여 사용자 인증.
    *   **회원가입**: 아이디, 패스워드, 이메일을 입력하여 신규 계정 생성.

2.  **고정지출 관리**
    *   **기본 등록**: 내역, 결제 항목(삼성카드, 우리은행 등), 고정지출 항목(구독, 월세 등), 결제일, 월 금액, 메모 입력.
    *   **추가 정보**: 할부나 렌탈 항목의 경우 기간, 총 금액, 금리 정보를 추가로 등록하여 관리.

3.  **대시보드 (Dashboard)**
    *   이번 달 총 고정지출 비용 합계.
    *   지출 완료된 금액 및 남은 지출 금액 현황.
    *   결제 항목별 지출 비중을 시각화한 통계 그래프.

4.  **설정 및 관리**
    *   **결제 항목 설정**: 사용자가 사용하는 카드사, 은행 등을 자유롭게 추가 및 수정.
    *   **지출 항목 설정**: 구독 서비스, 주거비 등 고정지출 카테고리를 커스텀하게 관리.

### UI 구성 (하단 탭 내비게이션)

*   **탭 1: 대시보드** - 지출 요약 정보 및 시각화 데이터 제공.
*   **탭 2: 고정지출 등록** - 새로운 고정지출 내역을 입력하는 메인 폼.
*   **탭 3: 설정** - 결제 항목 및 고정지출 항목 기준 정보 등록.
