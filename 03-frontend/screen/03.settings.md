# 03. 설정 및 관리 (Settings & Management)

## 1. 화면 개요
사용자의 계정 정보, 앱 환경 설정, 그리고 가계부 데이터(카테고리 등)를 관리하는 중앙 허브 화면입니다.
개인 프로필 수정부터 알림 설정, 데이터 초기화 및 로그아웃 기능을 제공합니다.

## 2. 주요 기능

### 2.1. 계정 관리 (Account Management)
-   **프로필 수정 (Edit Profile)**: 닉네임, 비밀번호 변경 기능을 제공합니다.
-   **로그아웃 (Logout)**: 현재 기기에서 로그아웃합니다.
-   **회원 탈퇴 (Delete Account)**: 계정과 모든 데이터를 영구적으로 삭제합니다.

### 2.2. 데이터 관리 (Data Management)
-   **카테고리 설정 (Category Settings)**: 수입/지출 카테고리를 추가, 수정, 삭제 및 순서 변경할 수 있습니다.
-   **데이터 내보내기 (Export Data)**: 가계부 내역을 엑셀(Excel) 또는 CSV 파일로 다운로드합니다.
-   **초기화 (Reset Data)**: 모든 거래 내역을 초기화합니다. (카테고리 유지 여부 선택 가능)

### 2.3. 앱 설정 (App Preferences)
-   **테마 설정 (Theme)**: 다크 모드, 라이트 모드 또는 시스템 설정을 따릅니다.
-   **알림 설정 (Notifications)**: 고정지출 예정 알림, 월별 리포트 알림 등을 켜고 끕니다.

## 3. UI 구성 요소 (Component Layout)

### A. 헤더 (Header)
-   좌측: 없음 (탭 메뉴 사용 시) 또는 뒤로가기 버튼
-   중앙: 화면 타이틀 ("설정")

### B. 설정 목록 (Settings List)
-   각 설정 항목은 그룹화(Section)되어 리스트 형태로 제공됩니다.
-   **Section 1: 내 정보 (My Info)**
    -   프로필 이미지 및 닉네임 표시
    -   `>` 화살표 아이콘으로 상세 페이지 진입 유도
-   **Section 2: 가계부 설정 (Ledger Settings)**
    -   카테고리 관리
    -   자산 설정 (통장, 카드 관리)
-   **Section 3: 일반 (General)**
    -   테마 설정 (Toggle 또는 Select)
    -   알림 설정 (Toggle)
    -   데이터 내보내기/초기화
-   **Section 4: 계정 (Account)**
    -   로그아웃 (텍스트 버튼)
    -   회원 탈퇴 (작은 텍스트, 붉은색)

### C. 하단 (Footer) - *Optional*
-   앱 버전 정보 표시 (예: v1.0.0)
-   이용약관 및 개인정보처리방침 링크

## 4. 데이터 요구사항 (Data Requirements)

### API: `GET /api/user/profile` (프로필 조회)
-   **Response**:
    ```json
    {
      "nickname": "알뜰이",
      "email": "user@example.com",
      "profileImage": "url/to/image.jpg",
      "theme": "system",
      "notificationEnabled": true
    }
    ```

### API: `PUT /api/user/settings` (설정 변경)
-   **Request Body**:
    ```json
    {
      "theme": "dark",
      "notificationEnabled": false
    }
    ```
-   **Response**: `200 OK`

### API: `GET /api/categories` (카테고리 목록)
-   **Response**: `[ { "id": 1, "name": "식비", "type": "EXPENSE" }, ... ]`

### API: `POST /api/auth/logout` (로그아웃)
-   **Response**: `200 OK`

## 5. 인터랙션 (Interactions)
1.  **화면 이동**: 리스트 아이템 클릭 시 해당 설정의 상세 페이지(Depth 2)로 슬라이드 이동합니다.
2.  **토글 스위치**: 알림, 테마 변경 등의 즉각적인 설정은 스위치(Toggle)로 제어하며, 변경 즉시 반영(API 호출)됩니다.
3.  **위험 동작 확인**:
    -   로그아웃: "로그아웃 하시겠습니까?" 확인 모달.
    -   초기화/탈퇴: 붉은색 경고 문구와 함께 "데이터를 복구할 수 없습니다." 재확인 모달(입력창을 통해 '삭제' 입력 등 안전 장치 고려).
4.  **카테고리 편집**: 드래그 앤 드롭으로 순서 변경, 스와이프로 삭제 기능을 제공할 수 있습니다.
