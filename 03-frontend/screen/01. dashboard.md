# 01. 대시보드 (Dashboard)

## 1. 화면 개요
사용자가 로그인 후 가장 먼저 접하는 화면으로, 이번 달의 고정지출 현황을 한눈에 파악할 수 있는 요약 정보를 제공합니다.
지출 총액, 납부 현황, 결제 수단별 비중 등을 시각화하여 사용자가 재정 상태를 직관적으로 이해하도록 돕습니다.

## 2. 주요 기능

### 2.1. 월별 조회 (Month Selector)
-   사용자는 조회하고자 하는 연/월을 선택할 수 있습니다.
-   기본값은 **현재 연/월**입니다.
-   이전/다음 달로 이동하는 화살표 버튼 또는 연/월 선택 팝업을 제공합니다.

### 2.2. 지출 요약 (Expense Summary)
-   **이번 달 총 고정지출**: 선택된 월의 고정지출 항목 총합계를 표시합니다.
-   **지출 완료 금액 (Paid)**: 이미 결제일이 지났거나 '지출 완료' 처리된 금액의 합계입니다.
-   **남은 지출 금액 (Remaining)**: 앞으로 지출 예정인 금액의 합계입니다.
-   **진행률 표시**: (지출 완료 금액 / 총 고정지출) 비율을 프로그레스 바(Progress Bar) 형태 등으로 시각화합니다.

### 2.3. 통계 그래프 (Visual Statistics)
-   **결제 수단별 지출 비중**:
    -   삼성카드, 우리은행 등 등록된 결제 수단별로 지출 금액이 차지하는 비중을 **도넛 차트(Donut Chart)** 또는 **파이 차트(Pie Chart)**로 표시합니다.
    -   각 섹션 클릭 시 해당 결제 수단의 상세 금액을 툴팁이나 범례로 표시합니다.

### 2.4. 임박한 지출 내역 (Upcoming Payments) - *Optional*
-   결제일이 다가오는 상위 3~5개 항목을 리스트 형태로 간략히 노출합니다.
-   항목명, 결제일(D-Day), 금액, 결제 수단을 표시합니다.

## 3. UI 구성 요소 (Component Layout)

### A. 헤더 (Header)
-   좌측: 앱 로고 또는 화면 타이틀 ("대시보드")
-   우측: 설정 아이콘 (탭 메뉴 외 별도 설정 진입 필요 시), 알림 아이콘

### B. 기간 선택 영역 (Date Navigator)
-   `<` [YYYY. MM] `>` 형태의 네비게이션
-   클릭 시 월 선택 모달 팝업

### C. 요약 카드 (Summary Cards)
-   **Total Expenses Card**: 
    -   Title: "이번 달 고정지출"
    -   Value: ₩1,250,000 (금액 강조)
-   **Status Card**:
    -   Row 1: "지출 완료" ₩800,000
    -   Row 2: "남은 금액" ₩450,000
    -   Visual: Progress Bar (64%)

### D. 차트 영역 (Chart Section)
-   Title: "결제 수단별 지출"
-   Chart: Pie/Donut Chart Component (e.g., Recharts, Chart.js)
-   Legend: 하단 또는 우측에 결제 수단 이름과 색상 표시

## 4. 데이터 요구사항 (Data Requirements)

### API: `GET /api/dashboard/summary`
-   **Parameters**: `year`, `month`
-   **Response**:
    ```json
    {
      "period": "2024-05",
      "totalAmount": 1250000,
      "paidAmount": 800000,
      "remainingAmount": 450000,
      "progressPercentage": 64.0,
      "paymentMethodStats": [
        { "name": "삼성카드", "amount": 500000, "color": "#123456" },
        { "name": "현대카드", "amount": 300000, "color": "#654321" },
        { "name": "우리은행", "amount": 450000, "color": "#abcdef" }
      ]
    }
    ```

## 5. 인터랙션 (Interactions)
1.  **월 이동**: 화살표 클릭 시 해당 월의 데이터로 새로고침(Refetch) 됩니다.
2.  **차트 터치**: 차트의 특정 영역 터치 시 해당 결제 수단의 구체적인 금액이 팝오버로 표시됩니다.
3.  **지출 내역 바로가기**: 요약 카드를 탭하면 상세 내역을 볼 수 있는 화면(또는 탭 2)으로 이동할 수 있습니다.
