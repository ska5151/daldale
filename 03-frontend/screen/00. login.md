# 로그인 화면*

## 기본 기능
1. **로그인**
   - 아이디 입력
   - 패스워드 입력
   - 로그인 버튼 (입력 완료 후 로그인 수행)

## 회원가입
1. **회원가입 버튼**
   - 버튼 클릭 시 회원가입 폼 제공

2. **회원가입 폼 입력 항목**
   - 아이디
   - 패스워드
   - 패스워드 확인
   - 이메일

3. **저장**
   - 모든 항목 입력 후 '저장' 버튼을 눌러 회원가입 완료
